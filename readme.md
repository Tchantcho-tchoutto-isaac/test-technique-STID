# test-technique-STID

# Playwright Cucumber TS Project

## Description
Ce projet est une suite de tests automatisés pour [ton application/site].  
Il utilise **Playwright**, **TypeScript** et **Cucumber** pour créer des tests end-to-end (E2E) robustes et maintenables.

---

## 📦 Technologies utilisées
- [Playwright](https://playwright.dev/) – Framework pour tests E2E
- [TypeScript](https://www.typescriptlang.org/) – Typage statique
- [Cucumber](https://cucumber.io/) – BDD (Behavior Driven Development)
- Node.js et npm

---

## Prérequis
- Node.js >= 18.x  
- npm ou yarn  
- Git (pour cloner le projet)

---

## 🚀 Installation

1. Cloner le repository :

```bash
git clone https://github.com/Tchantcho-tchoutto-isaac/test-technique-STID.git
cd Playwright_Cucumber_TS

```bash
2. Installer les dépendances:

npm install

3. Installer les navigateurs nécessaires pour Playwright :
npx playwright install
